# Trabalho final do curso de verão "Relatórios Reprodutíveis com R" do IME-USP, ministrado em 2026

Este relatório corresponde ao meu trabalho final do Curso de Verão “Relatórios Reprodutíveis com R”, ofertado pelo IME-USP. O projeto utiliza dados públicos brasileiros, obtidos a partir de bases oficiais, para explorar indicadores socioeconômicos por meio de análises descritivas e visualizações gráficas.

A análise foi desenvolvida integralmente em R, com organização do projeto e geração do relatório em Quarto, integrando texto, código, tabelas e gráficos de forma reprodutível
